Newsprint Recycling 
 
Accepted (All Paper should be Clean and Dry) 
 
• All Newsprint (If it comes with the Newspaper, it can be recycled with the Newspaper) 
• Phonebooks (with covers removed) 
 
Not Accepted 
 
• Wet/Soiled Newspaper 
• Plastic bags 
• Wax or Plastic-coated paper 
 
 
 
 
 
**IF YOU ARE NOT SURE ABOUT AN ITEM PLEASE ASK! **
